# Repository Structure

This document outlines the structure of the `CloudJourneyBlog` repository.

```
📁 CloudJourneyBlog/
├── 📁 content/           # Blog content, tutorials, and guides
│   ├── 📁 blogs/         # Categorized blog posts
│   └── 📁 guides/       # Comprehensive references
├── 📁 projects/         # Automation and tool projects
│   ├── 📁 azure-logic-apps/    # Logic Apps templates
│   ├── 📁 mac-os/              # Scripts for macOS automation
│   ├── 📁 n8n-workflows/       # A collection of n8n workflows for automation
│   └── 📁 sync-gal/            # A project for synchronizing the Global Address List (GAL)
├── 📁 infrastructure/   # Infrastructure templates
│   └── 📁 n8n-traefik-stack/ # A Docker Compose stack for n8n with Traefik
├── 📁 docs/            # Project documentation
│   └── 📁 architecture/      # Architectural documents
└── 📁 .github/         # GitHub-specific files
```
